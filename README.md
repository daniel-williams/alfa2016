# Anna Lancaster Fine Art

Public facing website for Anna Lancaster Fine Art. Asp.NET / Web API wrapper for sending email, otherwise, this is a ReactJS SPA.



## Contains
* React
* React-router
* Redux
* Bootstrap 3
* Masonry
* Firebase
* Babel
* Webpack


## Quick Start
Install dependencies
```bash
# using npm
$ npm i
```

Development
```bash
# browse to http:\\localhost:3000
$ npm start
```

Build
```bash
# output to /dist
$ npm run build
```
